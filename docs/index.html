[Incomplete]About

DesignsSquare.com Slider widget contains:

  • Elastic Slider: Simple slider that slides images
  • Revolution Slider: Slider that slides elements such text, images, videos with easy configuration of timing, slider effect, etc
  • more to come in future releases

[Incomplete]Installation

Here are steps to install Slider widget:

  1. Enable "DesignsSquare Slider Widget" for any slider to work

      For Elastic Slider

    1. Enable "DesignsSquare Elastic Slider Slide" which also will enable "DesignsSquare.com: Slider: Elastic: Structure"
    2. For importing Sample Data, enable "DesignsSquare.com: Slider: Elastic: Sample Data"
    3. Assign Elastic Slider block to a region you like the slider to be displayed. If sample data was imported, then there is already block "DesignsSquare.com: Slider: Elastic" to be assigned. Otherwise, create a view with tag - designssquare_slider_elastic as described in the "How To" section below for the Elastic Slider

      For Revolution Slider

    1. Enable "DesignsSquare Rev Slider" which also will enable "DesignsSquare.com: Slider: Revolution: Structure"
    2. For importing Sample Data, enable "DesignsSquare.com: Slider: Revolution: Sample Data"
    3. Assign Rev Slider block to a region you like the slider to be displayed. If sample data was imported, then there is already block "DesignsSquare.com: Slider: Rev Slider" to be assigned. Otherwise, create a view with a tag - designssquare_slider_rev_slider as described in "How To" for Revolution Slider

[Incomplete]How To

Setup Elastic Slider

To setup Elastic Slider:

  1. Create an instances of the type 'Elastic Slider' by going to 'Content'->'Add'->'Elastic Slider'
  2. Create a view containing instances of Elastic Slider that you wish be part displayed in slider. Afterwards, assign a tag - "designssquare_slider_elastic" to this view.
  3. Assign the view to the region you wish the slider be rendered

Setup Revolution Slider

To setup Elastic Slider:

  1. Create an instance of the type 'Revolution Slide'. Do it for each slide
  2. Create a view containing the Slides you like to be in the Slider and assign tag "designssquare_slider_rev_slider"
  3. Assign this view to the region

Configure Sidebar

Each slider can have a sidebar. To configure sidebar go to "admin/designs-square/slider"

A Word on Elastic Slider Sampla Data

If you imported sample data for the Elastic Slider, then you will have two Elastic sliders - Elastic Slider Sample - DesignsSquare.com, Elastic Slider Sample - Bazar. Since only one slider is rendered at the time, the slider Elastic Slider Sample - DesignsSquare.com is going to be rendered. You can switch to the Elastic Slider Sample - Bazar by making Elastic Slider Sample - DesignsSquare.com unpublished. This is because the view(i.e.DesignsSquare.com: Slider: Elastic Slider) controlling the Elastic Slider filters published content. In another words, you can do much of what to display for Elastic Slider by configring the view - DesignsSquare.com: Slider: Elastic Slider

Designssquare Rev Slider Layer
